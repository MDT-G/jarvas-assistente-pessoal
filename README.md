# JARVAS - Assistente Pessoal de Finanças 

O JARVAS é um projeto desenvolvido em Python para automatizar o controle financeiro e o planejamento de reforma do meu primeiro imóvel (entrega prevista para 2027).

## Funcionalidades
- **Gestão de Boletos:** Cadastro e visualização de contas vinculadas ao imóvel.
- **Lista de Desejos:** Planejamento de itens para a reforma com cálculo de soma total automática.
- **Persistência de Dados:** Uso de arquivos JSON para armazenamento seguro.
- **Integração Temporal:** Saudações automáticas e controle de prazos via biblioteca `datetime`.

##  Contexto
Este projeto faz parte dos meus estudos práticos de Python, aplicando conceitos de manipulação de arquivos, lógica de programação e tratamento de dados.

##  Como rodar
1. Clone o repositório.
2. Execute o arquivo `jarvas.py`.
